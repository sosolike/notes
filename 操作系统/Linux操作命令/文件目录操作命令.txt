目录操作：
    1：创建目录(mkdir)：
        mkdir [-m 权限值][-p] 待创建目录路径
        功能：创建目录路径
        参数：-m：指定该目录的权限值(默认为系统默认权限)
             -p：递归创建目录(mkdir 本身不能递归创建目录)
        常用使用方式：
            1：创建一个目录，并设置为系统默认权限(/dir1必须已经存在)：
                mkdir /dir1/dir2 
            2：创建一个目录，并设置该目录权限(dir1必须已经存在)：
                mkdir -m 761 /dir1/dir2
            3：递归创建一个目录：
                mkdir -p /dir1/dir2

    2：删除目录(rm/rmdir)：
        rm -r[f]
        功能：递归删除指定目录
        参数：f：强制删除，不再询问
        使用方式：
            1：删除目录路径并询问：
                rm -r 待删除目录路径
            2：删除目录路径，不再询问：
                rm -rf 待删除目录路径
        rmdir [-p]
        功能：递归删除指定空目录
        参数：-p：递归删除空目录
        使用方式：
            1：删除指定空目录(待删除目录必须是空路径)：
                rmdir 目录路径
            2：如果子目录被删除后，父级目录也是空目录，则将空目录一并删除：
                rmdir -p 目录路径
    3：复制目录(cp)：
        cp [-dpRafil] 待复制目录路径 目标目录路径
        功能：cp本身用来进行文件或者目录的复制
        参数：d：复制时保留链接
             p：除过复制文件内容以外，还把修改时间和访问权限也复制到新文件中
             R：复制目录
             a：等同于同时使用-dpR
             f：无论文件还是目录，强制复制，如果目标文件/目录已经存在，也强制覆盖；
             i：如果目标文件/目录已经存在，则将询问是否继续进行复制操作；
             l：不复制文件，只生成链接文件；
        
    4：移动目录(mv)：
        mv [-ifu] 源目录路径 目标目录路径
        功能：移动目录或者文件
        参数：i：如果目标文件已经存在将会发起询问是否继续移动操作
             f：强制移动，不再询问是否覆盖已经存在的目标文件；
             u：如果目标文件已经存在，但是比源文件修改时间更早则会移动当前文件覆盖旧文件；

    5：修改目录权限(chmod)：
        chmod [-R] 权限值 待修改路径
        功能：将目标目录权限修改至指定权限
        参数：对目录递归设置权限(默认只对指定目录设置权限，带上R参数之后将会对目录中的子目录等设置同样的权限)
        权限：读：4、写：2、执行：1；权限值又包括了目录所属用户、目录所属分组用户、其他用户权限；
        常用使用方式：
            1：将目标目录修改为777权限：
                chmod -R 777 ./test
    
    6：修改目录所属用户(chown)：
        chown [-R] 所属用户名/所属用户id 待修改路径
        功能：将目标目录所属用户修改至指定用户
        参数：对目录递归设置所属用户(默认只对指定目录设置所属用户，带上R参数之后将会对目录中的子目录等设置同样的所属用户)
        所属用户：可以是用户名称或者用户id
        常用使用方式：
            1：将目标目录修改为abc用户：
                chown -R abc ./test

    7：修改目录所属分组(chgrp)：
        chgrp [-R] 所属分组名/所属分组id 待修改路径
        功能：将目标目录所属分组修改至指定分组
        参数：对目录递归设置所属分组(默认只对指定目录设置所属分组，带上R参数之后将会对目录中的子目录等设置同样的所属分组)
        所属分组：可以是用分组名称或者分组id
        常用使用方式：
            1：将目标目录修改为abc分组：
                chgrp -R abc ./test

    8：查看目录路径(pwd)：
        pwd [-P]
        功能：查看当前目录路径
        参数：P：显示目录链接的真实路径
        常用使用方式：
            1：查看本目录路径
                pwd
            2：如果当前目录为另一个目录的链接，查看方式：
                pwd -P
            
    9：进入目录(cd)：
        cd 目录路径
        功能：进入目录
        参数：无
        常用使用方式：
            1：进入本级目录下的子目录dir：
                cd ./dir   
            2：进入上级目录：
                cd .. 或者 cd ../
            3：进入根目录：
                cd /
            4：进入当前用户home目录：
                cd ~
            5：进入指定目录地址：
                cd /dir1/dir2/dir3

   10：查看目录信息(ls/ll)
        ls [-alrtAFR] [name]
        功能：查看当前目录列表
        参数：
            a：查看当前目录中所有子目录和文件,包括隐藏文件和.以及..目录(默认只显示非隐藏文件以及除过.和..之外的目录文件)
            l：以详细模式展示目录中的子目录和文件信息，详细信息包括文件大小、名称、权限、修改日期等(默认只显示文件名称)
            r：将文件目录以相反次序显示(默认以英文字母顺序排序)
            t：将文件以建立时间顺序列出(默认以英文字母顺序排序，rt参数同时带上，则以建立时间反向排序)
            A：和a功能类似，但是不列出.和..目录
            R：递归列出目录,即列出子目录中的文件(默认不会递归列出)
        常用使用方式：
            1：查看本目录下的文件和子目录：
                ls
            2：查看本目录下的文件和子目录并列出详细信息:
                ls -l
            3：查看本目录下所有文件和子目录并列出详细信息：
                ls -al
            4：查看本目录下所有文件和子目录，列出详细信息，并排序：
                ls -alt 或者 ls -alr 或者 ls -alrt
        ll：类似于ls -l，同样也可以使用上述参数来调整ll的输出信息；

    11：查看目录/文件占用磁盘大小(du)：
        du [-ashclL] 文件/目录
        功能：查看文件/目录占用的磁盘空间大小，该大小并非根据文件实际内容占用的字节数统计而出，而是根据文件占用的磁盘block数量计算而出；
        参数： 
            -a：显示指定目录占用磁盘大小，并显示该目录下子目录和文件占用磁盘空间大小；
            -s：显示指定目录占用磁盘大小时，不要显示该目录下子目录和文件占用的磁盘空间大小；
            -c：显示几个文件、目录占用磁盘空间的大小，并统计他们的总和；
            -h：以可读的方式显示；即带上大小单位如：GB、KG、MB
            -l：统计硬链接占用磁盘空间大小；
            -L：统计符号链接所指向的文件占用磁盘空间大小；
        说明：
            1：du命令检测的文件大小实际是文件占用的磁盘block数量*磁盘block大小；linux下默认一个block大小是4kb；
            2：如果查看文件实际字节数，需要使用ls -l或者ll来查看；
        常见使用方式：
            1：显示当前目录占用磁盘空间大小,并统计子目录占用磁盘空间大小：
                du -ah ./
            2：显示当前目录占用磁盘空间大小，不显示该目录下子目录占用磁盘空间的大小：
                du -sh ./
            3：显示当前目录占用磁盘空间大小，并计算总和：
                du -ch ./

    12：关于目录操作的其他常用命令：
        1：查看目录中文件数：
            ls -l |grep "^-"|wc -l
            说明：通过检测文件类型为-来判断文件类型
        
        2：查看目录中的目录数：
            ls -l |grep "^d" |wc -l
        
        3：递归查看目录中的文件数/目录数：
            文件数：ls -lR |grep "^-"|wc -l
            目录数：ls -lR |grep "^d"|wc -l
